## BrainRot GPT

Turn PDFs into short form, digestible videos. Perfect if you have a sub-30 second attention span.

## User Guide
- Login
- Upload PDF
- Pick video type (ie minecraft or subway surfers)
- Wait 1-1.5 mins while it generates
